【裁判字號】 91,易,860
【裁判日期】 911212
【裁判案由】 違反槍砲彈藥刀械管制條例
【裁判全文】   
臺灣臺中地方法院刑事判決　　　　　　　　　　　　　九十一年度易字第八六號
　　公　訴　人　台灣台中地方法院檢察署檢察官
　　被　　　告　甲○○
　　　　　　　　乙○○
右列被告因違反槍砲彈藥刀械管制條例案件，經檢察官聲請以簡易判決處刑（九十年
度偵字第二二０六九號），本院台中簡易庭認為不應適用簡易程序，移送刑事庭改依
通常程序審理，判決如左：
    主  文
甲○○、乙○○均無罪。
    理  由
一、公訴意旨略以：被告甲○○、乙○○均明知未經許可，不得轉讓、持有魚槍。甲
    ○○竟於不詳時間，以不到新台幣（下同）七千元之價格，向位在高雄市三民區
    ○○○路六三八巷十九號之「大順潛水器材社」購買未經許可之魚槍一支，再於
    民國九十年十一月二十八日，在其所經營位於臺中市○○路○段四六一號之「珊
    瑚礁潛水樂園」內，以七千元之價格，將該魚槍轉讓予知情之乙○○。詎乙○○
    取得該魚槍後，未申請許可，即將之收藏在其位於台中縣大肚鄉○○○街五八號
    之住處而持有，至九十年十二月六日凌晨一時許，為警在其上址住處之倉庫內扣
    得該支魚槍，進而查獲上情，因認被告甲○○犯有槍砲彈藥刀械管制條例第九條
    第一項之未經許可轉讓魚槍罪嫌；被告乙○○則犯有同條例同法條第三項之未經
    許可持有魚槍罪嫌。
二、按犯罪事實應依證據認定之，無證據不得推定其犯罪事實，又不能證明被告犯罪
    者，應諭知無罪之判決，刑事訴訟法第一百五十四條、第三百零一條第一項定有
    明文。又最高法院三十年上字第八一六號判例意旨謂認定不利被告之事實，須依
    積極證據，苟積極證據不足為不利於被告事實之認定時，即應為有利被告之認定
    。另最高法院七十六年臺上字第四九八六號判例意旨稱認定犯罪事實所憑之證據
    ，無論其為直接證據或間接證據，其為訴訟上之證明時，均須達於一般之人不至
    有所懷疑，而得確信其為真實之程度，始得據為有罪之認定，倘其證明尚未達到
    此一程度，而有合理性之懷疑存在時，即無從遽為有罪之確信。
三、公訴人認定被告分別涉有前述違反槍砲彈藥刀械管制條例之罪嫌，無非係以被告
    於偵查中均坦承知悉未經主管機關許可，不得轉讓或持有魚槍，及扣案之魚槍經
    送請內政部警政署刑事警察局鑑定之結果，認係法國製市售魚槍，以橡皮彈力為
    動力，機械性能良好，可發射隨附之金屬箭，具殺傷力，有該局九十年十二月十
    七日刑鑑字第二三○三五七號鑑驗通知書一紙在卷可憑。惟被告於本院審理時，
    雖供述確有轉讓或持有該支魚槍之行為，然均堅決否認前開罪嫌，分別辯解如左
    ：
    (一)被告甲○○辯稱：「當初我向大順潛水器材社購買這支魚槍時，對方有提供魚
      槍查驗證，我因此認為這是可以合法買賣，我並不知購買後，竟然會觸犯槍砲
      彈藥刀械管制條例，我也不知買賣這支魚槍，事前或事後，應否先向主管機關
      或警察機關申請許可，或事後報備」、「他（指乙○○，下同）於九十年十一
      月二十八日晚上向我購買這支魚槍，我告訴他要申請乙種自衛槍枝許可證，並
      必須拍魚槍照片二張，我有將申請書交給他，並交代他要找三個保證人，且必
      須沒有前科，然後向管區警員辦理。我知道要持有這支魚槍必須申請許可」、
      「他於二十九日早上打電話告訴我，這支魚槍於二十八日晚上遺失，他於二十
      九日已向警局備案」等語。
    (二)被告乙○○辯稱：「我確實是在九十年十一月二十八日，在珊瑚礁潛水樂園，
      以七千元的價格，購買這支魚槍。我購買時甲○○有告訴我要申請許可，所以
      我買了之後就先拍照，之後就放在我的住處，後來當晚魚槍遺失，我於二十九
      日下午六、七時許向警方申報遺失」、「他（指甲○○）當時有拿了申請書給
      我，並告訴我必須拍照，並找保證人後，向當地管區警員辦理申請許可，所以
      我在當晚七點多，就趕著去辦理，不過魚槍在當晚就遺失了，我於十一月二十
      九下午六至七點之間，向警局報案後，警員直到十二月六日晚上才到我家調查
      ，要拍失竊現場的照片，我母親見狀，才把槍拿出來，此時我才知道，我媽是
      怕我發生危險，所以在二十八日晚上我將魚槍買回來後，她就藏起來了」等語
      。
四、經查：
    (一)本案明顯涉及合法經營之體育用品社、魚具店、潛水器材社等未申請許可，得
      否販賣魚槍，及一般民眾欲合法購買擁有魚槍者，應如何辦理等問題。此經內
      政部以九十一年九月二十四日內授警字第０九一００七六四０六號函覆本院，
      說明如左，本院認為尚無不合：
      1.「按本部八十五年三月十二日內警字第八五七０三０一號函頒『槍彈使用
        許可及管理作業規定』貳－九規定：經核准產製或進口魚槍之廠商，於每案
        生產或進口前，將數量、型號、型錄報請內政部許可後，始得製造或進口；
        裝配完成或通關進口後，應報由當地警察機關實施查驗後，准於合法經營之
        體育用品社、魚具店、潛水器材社等陳列、販賣，免申請販賣許可。是以，
        上開商店陳列、販賣經主管機關查驗核發查驗證之魚槍，免申請販賣許可」
        。
      2.「一般民眾欲購買擁有魚槍者，目前實務上可先至前開商店購買，並即持向
        戶籍所在地警察機關申請核發魚槍執照，執照未核發前魚槍一律不得使用（
        如附魚槍查驗證影本）。又即將訂定發布之『槍砲彈藥刀械許可及管理辦法
        （草案）』第八條及第十條（如附草案），對魚槍之購置許可使用，已有明
        確規定」。
    (二)又前述之魚槍查驗證，依內政部該函所附影本，格式如左：
      ┌─┬───────┬────┬────────┐
      │魚│製  造  廠  商│製造廠牌│槍  枝  號  碼  │
      │  ├───────┼────┼────────┤
      │槍│              │        │                │
      │  │              │        │                │
      │查│              │        │                │
      │  ├──┬────┴────┴────────┤
      │驗│查驗│內政部警政署                        │
      │  │機關│                                    │
      │證├──┼──────────────────┤
      │第│發證│中華民國      年      月          日│
      │  │時間│                                    │
      │  ├─┬┴──────────────────┤
      │  │注│一、本證為申請核發槍枝執照之憑證。    │
      │  │意│二、槍枝購用人應即向戶籍所在地警察機關│
      │  │事│    申請核發執照。執照未核發前，魚槍一│
      │  │項│    律不得使用。                      │
      │號│  │                                      │
      └─┴─┴───────────────────┘
    (三)而被告甲○○前開否認犯行之辯解，已提出扣案魚槍之魚槍查驗證影本附卷，
      詳如左列之形式，此張魚槍查驗證之反面，並有高雄巿政府警察局之魚槍查驗
      章：
      ┌─┬───────┬────┬────────┐
      │魚│製  造  廠  商│製造廠牌│槍  枝  號  碼  │
      │  ├───────┼────┼────────┤
      │槍│大順潛水器    │帆船牌  │Ｅ２０１７１    │
      │  │材有限公司    │        │                │
      │查│              │        │                │
      │  ├──┬────┴────┴────────┤
      │驗│查驗│內政部警政署                        │
      │  │機關│台灣省政府警務處                    │
      │證├──┼──────────────────┤
      │第│發證│中華民國捌拾陸年  陸  月  貳拾伍  日│
      │１│時間│                                    │
      │２├─┬┴──────────────────┤
      │４│注│一、本證為申請核發槍枝執照之憑證。    │
      │１│意│二、槍枝購用人應即向戶籍所在地警察機關│
      │１│事│    申請核發執照。執照未核發前，魚槍一│
      │８│項│    律不得使用。                      │
      │號│  │                                      │
      └─┴─┴───────────────────┘
    (四)經參酌前述內政部八十五年三月十二日內警字第八五七０三０一號函頒「槍
      彈使用許可及管理作業規定」貳－九之規定，及被告甲○○所提出扣案魚槍之
      「魚槍查驗證」，顯見被告甲○○將扣案之魚槍販賣給被告乙○○之行為，並
      不該當槍砲彈藥刀械管制條例第九條第一項之未經許可，轉讓魚槍罪。
    (五)一般民眾欲購買擁有魚槍時，可先至前開商店購買，並即持向戶籍所在地警察
      機關申請核發魚槍執照，執照未核發前魚槍一律不得使用；尚無須於購得魚槍
      前，先行報請許可，已見前述。故被告乙○○向甲○○買受後持有該支魚槍之
      行為，並不當然構成槍砲彈藥刀械管制條例第九條第三項之未經許可，持有魚
      槍罪。而被告乙○○向甲○○購買該支魚槍，經被告甲○○告知應即辦理申請
      許可之事項，被告乙○○隨即於取得魚槍之當日下午著手辦理，先行拍照，不
      久後發覺遺失，而於翌日傍晚約六、七時許報警處理等過程，不僅被告二人於
      本院審理時，經隔離訊問，所述相符，有如前開三之(一)、(二)所載；且參本件偵
      查卷內，確實附有被告乙○○於九十年十一月二十九日報案後，為警於當日調
      查詢問之警訊筆錄一份（見偵查卷第九－十頁）；再者，警方於九十年十一月
      二十九日對被告甲○○調查詢問時，被告甲○○亦有言及被告乙○○取得向其
      購買之魚槍後即至相館拍照，並持申請書欲自行辦理申請許可等語（見偵查卷
      第十一頁背面倒數第三、四行所載）。從上述情節不難索解被告乙○○取得魚
      槍後，確即開始進行申請許可之必要事項，後來因認魚槍遺失，乃未繼續辦理
      ，其主觀上顯無槍砲彈藥刀械管制條例第九條第三項未經許可持有魚槍罪之犯
      意，自無此項罪嫌可言。
五、綜合前述，被告二人之罪嫌，顯有不足。既不能證明其等確有公訴人所指違反槍
    砲彈藥刀械管制條例之犯行，即應諭知均無罪之判決，以符法制。
據上論斷，應依刑事訴訟法第四百五十二條、第三百零一條第一項，判決如主文。
中      華      民      國      九十一    年    十二    月      十二    日
                                        法  官  莊  深  淵
右正本證明與原本無異。
如不服本判決，應於送達後十日內向本院提出上訴狀（須附繕本），上訴於台灣高等
法院台中分院。
                                        書記官
